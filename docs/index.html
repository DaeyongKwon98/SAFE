Step-level feedback for reliable multi-hop reasoning

Abstract

Reasoning is treated as a sequence of verifiable decisions.

SAFE generates one attribution, logical inference, or final-answer step at a time. A LoRA evaluator checks grounding, relevance, novelty, efficiency, logical validity, and answer timing. Rejected steps are regenerated from a diagnosis and immediate guidance instead of allowing an early error to propagate.

Method

One implementation, three controlled modes

01

Baseline

The generator advances without external verification and stops at a final answer.

02

Self-feedback

The generator evaluates its own proposed step and retries rejected steps.

03

SAFE

A separately trained PEFT evaluator supplies targeted revision guidance.

Reproduction

Transformers only. No hosted model API is required.

pip install -r requirements.txt
python -m safe prepare --config configs/main.yaml \
  --benchmark examples/benchmark.jsonl \
  --output outputs/data/evaluator.jsonl
python -m safe reproduce --config configs/main.yaml --dry-run

The reference setup uses Python 3.10, CUDA 12.8, 4-bit NF4 loading, PEFT adapters, and automatic multi-GPU placement.
